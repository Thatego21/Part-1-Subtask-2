Fryer Oil Temperature Monitoring Program

🔹 Purpose
This program helps monitor fryer oil temperature at Magwinya Magic. It compares three temperature readings and gives instructions to adjust the heat or start frying when the oil is ready.

🔹 Files Included
- fryer_temperature.cpp — main C++ source code  
- images/ipo_chart.png — IPO chart showing Inputs, Processing, and Outputs  
- images/flowchart.png — flowchart diagram of program logic  

🔹 How It Works
1. The user enters three temperature readings.  
2. The program checks if the second reading increased by more than 50 or less than 10 degrees.  
3. It then checks if the third reading is between 150 and 190 degrees.  
4. Based on these checks, it displays messages such as:  
   - “Reduce fryer heat”  
   - “Increase fryer heat”  
   - “You may start frying”  
   - “Oil is not ready”
